## Installing
## Prerequisites
  • Nodejs- v18.14.1 or higher.
  • Text Editor - Recommended: Vscode
1. Setting up Vader . Before you can create your webapp you need to install the vaderjs npm package or use a cdn
npm install vaderjs
or

Use cdn version - this does not include vscode jsdoc types and is not recommended for production use

<script src="https://cdn.jsdelivr.net/npm/vaderjs/index.js"> </script>

Then you can

import Vader, { include } from "vaderjs"
