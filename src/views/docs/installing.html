## Installing
## Prerequisites
  • Nodejs- v18.14.1 or higher.
  • Text Editor - Recommended: Vscode
1. Setting up Vader

Before you can create your webapp you need to install the vaderjs npm package or use a cdn


npm install vaderjs
or

Use cdn version - this does not include vscode jsdoc types and is not recommended for production use

<script type="module" src="https://cdn.jsdelivr.net/npm/vaderjs@latest/index.js"> </script>
<script type="module" src="https://unpkg.com/vaderjs@latest/index.js"> </script> 2. Setting Up Import Maps

Import maps allow you to import modules without having to use relative paths


 <script type="importmap">  {
        "imports":{
            "vaderjs":"./dist/vader/index.js",
         }
     }
 </script>

Then you can

import Vader from "vaderjs"
