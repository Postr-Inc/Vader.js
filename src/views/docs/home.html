# # What is Vader? Vader.js is a reactive framework for single application web apps similar to react.js

Vader.js is a vanilla js framework that allows you to create components with a class based syntax it brings reactivity and templating to your workflow in a simple and easy to use manner.

# # Key Features

  • Class Based Components: - vader leverages the power of js classes to maintain both structue and state of components

  • Reactivity: - vader offers several state hooks like signals to add reactivity to your webpage
  • Templating: vader's html method extends html's core functionality
  • Routing - vader offers a router that allows you to create single page applications with ease
  • Lightweight - vader is a lightweight framework that is only 3kb gzipped
